# 登录注册系统

一个现代化的、响应式的登录注册页面，具有完整的表单验证和用户交互功能。

## 功能特性

### 1. 用户界面
- **现代化设计**: 采用渐变背景和卡片式布局
- **响应式布局**: 适配各种屏幕尺寸（桌面、平板、手机）
- **双面板设计**: 左侧为表单区域，右侧为欢迎信息面板
- **图标集成**: 使用Font Awesome图标增强用户体验

### 2. 登录功能
- 用户名/邮箱登录
- 密码显示/隐藏切换
- "记住我"选项
- 忘记密码功能
- 社交登录（Google、GitHub）

### 3. 注册功能
- 用户名、邮箱、密码、确认密码输入
- 实时密码强度检测
- 服务条款同意选项
- 自动切换到登录功能

### 4. 表单验证
- 实时输入验证
- 必填字段检查
- 邮箱格式验证
- 密码强度验证
- 密码一致性检查

### 5. 交互功能
- 登录/注册表单切换
- 密码显示/隐藏
- 表单提交模拟
- 通知系统
- 键盘快捷键支持

### 6. 技术特性
- 纯HTML/CSS/JavaScript实现
- 无外部依赖（除图标和字体）
- 模块化JavaScript代码
- 本地存储支持（记住我功能）

## 文件结构

```
public/
├── login.html          # 主HTML文件
├── style.css          # 样式文件
└── script.js          # JavaScript交互文件
```

## 使用方法

### 1. 本地运行
```bash
# 进入项目目录
cd public

# 启动HTTP服务器
python3 -m http.server 8000

# 在浏览器中访问
# http://localhost:8000/login.html
```

### 2. 部署到Web服务器
将 `public` 目录中的所有文件上传到您的Web服务器即可。

## 功能演示

### 登录功能
1. 输入用户名/邮箱和密码
2. 点击"登录"按钮或按Enter键
3. 系统会模拟登录过程并显示成功消息

### 注册功能
1. 点击"立即注册"链接切换到注册表单
2. 填写所有必填字段
3. 观察密码强度指示器
4. 同意服务条款
5. 点击"注册"按钮

### 键盘快捷键
- **Ctrl+L**: 切换到登录表单
- **Ctrl+R**: 切换到注册表单
- **Esc**: 清除所有表单内容

### 社交登录
- 点击Google或GitHub按钮体验社交登录流程
